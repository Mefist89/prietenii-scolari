# Prietenii Școlari (School Friends)

An interactive educational game for children to learn Mathematics, Computer Science, and Romanian Language through fun adventures with friendly characters.

## Characters

- **Maya** - Mathematics teacher
- **Byte** - Computer Science teacher
- **Lumi** - Romanian Language teacher

## Features

- 3 educational subjects with 5 levels each
- Final boss battles for each subject
- Child-friendly interface with colorful design
- Multi-language support (Romanian / English)

## Tech Stack

- [Next.js](https://nextjs.org) - React framework
- [Tailwind CSS](https://tailwindcss.com) - Styling
- [Framer Motion](https://www.framer.com/motion/) - Animations

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

```
app/
├── page.tsx              # Home page
├── matematica/           # Mathematics levels
├── informatica/          # Computer Science levels
├── limba-romana/         # Romanian Language levels
├── contacte/             # Contact page
├── creatori/             # Creators page
└── cum-sa-joci/          # How to play page
```
